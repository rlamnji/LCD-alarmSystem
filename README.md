# LCD-alarmSystem
> 임베디드시스템 TeamProject : 아두이노기반 LCD 알람 시스템


## Description
가변저항을 사용하여 알람 시간을 설정하고, 설정된 알람 시간과 현재 시간을 LCD 화면에 출력하며, 알람 시간이 되면 알람이 울리는 시스템을 구현한다.

## 설치 방법 및 사용 예제
* Arduino IDE 설치
* 라이브러리 설치
  + #include <LiquidCrystal_I2C.h>


## 하드웨어
* Arduino Uno R3
* 가변저항
* 피에조 스피커
* 버튼 2
* LCD 16x2(I2C)
